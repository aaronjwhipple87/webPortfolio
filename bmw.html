BMW M Class Enthusiast

The purpose of this project was to be introduced to the use of mockups to help in planning and creating projects more effectively. This project was my first time using a mockup software to stage and plan a website. Prior to the use of mockups it was very difficult and frustrating to change things after coding a site. The hardest part with this project was learning how to use Adobe software, but have since noticed how using the mockup software is essential in the planning stage.

Software/ Language

WebStorm, HTML5, CSS3, Boostrap, Adobe XD

Time
